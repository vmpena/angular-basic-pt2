Nesting Components

First, create both parent and child components. The child component uses the selector app-child so we include that element inside the parent template. The variable inside the parent component that will be passed to the child component is called searchTerm: string; You then map that variable to the child component template like this: <app-child [searchTerm]="searchTerm">
