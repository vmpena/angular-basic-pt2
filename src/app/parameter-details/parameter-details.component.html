Parameter Details

The route for this page has no configuration to handle incoming parameters. The parameters are handled via the Angular ActivatedRoute service.
The route is: path:'parameter-details', component: ParameterDetailsComponent The template handles the incoming route values just like properties of an object bound to this component. So the line below is written as: The code for {{city}} is {{code}} where the values between curly braces are substituted for the values passed in via the route.
